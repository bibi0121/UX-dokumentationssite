Features og funktionaliteter

AOF:

Aktiviteter:
  • Lektiehjælp online.
  • Ofte stillede spørgsmål.
  • Kan tjekke dagens/ugens skema.
  • Events/kurser.
Objekter:
  • Chatte/gruppechat, være i kontakt med andre studerende via appen.
  • Like opslag/spørgsmål og svar.
  • Privat snak, med en studerende.
  • Dele opslag/spørgsmål og svar.
  • Man plejer sin egen profil og holder den opdateret, med for eksempel færdigheder.
  • Give feedback til hinanden.
Features:
  • Offline mode: Man kan se Ofte stillede spørgsmål, når man er offline.
  • Indsætte/oploade filer og billeder.
  • Notifikationer, når man for eksempel for svar på en besked, eller et spørgsmål.
  • Kalender.
  • Chatfunktion.
  • En tilkoblet mail.

Brugssituation:

  1. Download appen KEA-hjælper.
  2. Log ind med uni-login.
  3. Opret bruger/profil.

Opnå med appen:

  • Lektiehjælp, både online og offline.
  • Let tilgængelig skema.
  • Information om forskellige evntes og kurser som foregår på kea, eller er passende til de studerende.
  • Kontakt med medstuderende, både nationalt og internationalt.
  • Man kan læse om de forskellige studerende på deres profil. Man kan læse/se færdigheder, hvilket semester de går på, chatte med dem privat og se deres profilbillede
